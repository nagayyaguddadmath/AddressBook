This application is to read given text file and it finds:
How many males are in the address book?
Who is the oldest person in the address book?
How many days older is Bill than Paul?

Few Observation:
1) This project needs Java 1.8 version to run
2) Main class to run is "Application.java" and thsi has all the methods to display the required output
3) We can pass input file path as an argument and by default it will take input file present in resource folder
4) JUNIT test class is "AddressBookTest.java". This has both positive and 2 negative test cases. 
4) We can run this project either from main class or from JUNIT test classes 
5) For simplicity sake , i have kept default input file in resources folder
